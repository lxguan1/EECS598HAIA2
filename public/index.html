Image Labeling Task

This is a study regarding the annotation of 2D images. In this task:
  • You will be presented a total of 5 images, each one paired with a machine label and a question.
  • The questions are yes or no questions, and can be answered by selecting the appropriate button.
  • The machine label is a reference for you, it is not necessarily always accurate
  • Press the 'submit' button after you have carefully completed all of the questions to receive an unique code you can enter into the MTurk survey to get compensation.
